Virtual Try-On Widget

Embed this widget on any website with a single line of code

Quick Start

Step 1: Add this script tag to your HTML:

<script src="https://your-domain.vercel.app/widget.js"></script>

Step 2: Add this div where you want the widget to appear:

<div id="vton"></div>

Alternative - Multiple Widgets: Use data attribute:

<div id="widget-1" data-vton-widget></div>
<div id="widget-2" data-vton-widget></div>
Note: The widget will automatically initialize when the page loads. No additional configuration needed!

Setup Instructions

1. Get Hugging Face Token

Go to https://huggingface.co/settings/tokens and create a token.

2. Add Environment Variables

Create a .env file in your project root:

HF_TOKEN=your_token_here
PORT=3000
API_URL=http://localhost:3000

3. Install Dependencies

npm install

4. Run the Server

npm run dev

5. Open in Browser

http://localhost:3000

Deploy to Vercel

Push your code to GitHub and deploy:

  1. Connect your GitHub repo to Vercel
  2. Add HF_TOKEN to Environment Variables in Vercel dashboard
  3. Set API_URL to your Vercel domain (e.g., https://your-app.vercel.app)
  4. Deploy and enjoy!
