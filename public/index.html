Virtual Try-On Widget

Embed this widget on any website with a single line of code

Quick Start

Step 1: Add this script tag to your HTML:

<script src="https://your-domain.vercel.app/widget.js"></script>

Step 2: Add this div where you want the widget to appear:

<div id="vton"></div>
Note: The widget will automatically initialize when the page loads!
